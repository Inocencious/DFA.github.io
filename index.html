What is RegEx?

Regular expressions (RegEx) are compact and powerful notation used to describe sets of patterns and strings in automaton theory. They provide a way to define regular languages, a class of languages ​​that can be recognized by a finite automaton.

What is DFA?

A deterministic finite automaton (DFA) is a mathematical model used in automaton theory and formal languages ​​to recognize or generate strings in a language. It is a kind of finite state machine that operates deterministically. There is only one transition to the next state for a given input symbol.

What is CFG?

Context-free grammar (CFG) is formal grammar used to describe the syntax or structure of a language in automaton theory and formal language theory. It provides a set of production rules that define how symbols are combined to form valid strings within the language.

What is PDA?

A pushdown automaton (PDA) is a type of abstract computational model in automata theory. It extends the capabilities of a finite automaton by incorporating a stack, which allows for more complex and powerful computations. PDAs are commonly used to recognize context-free languages.
